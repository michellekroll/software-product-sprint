BearMaps

Implemented a web browser mapping application similar to Google Maps using the A* search algorithm and utilized data structures such as KDTrees, Heaps, and Priority Queues.

Build Your Own World

Worked in collaboration with one other student to create a 2D tile-based random world exploration engine, which involved implementing UI with Heads Up Display, keyboard mode, and string mode where users can interact with a moving avatar. Utilized KDTrees and A* search algorithm to create paths for avatar to navigate randomly generated rooms and hallways in the world.
